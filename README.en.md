# AI Knowledge Base

AI Knowledge Base is an LLM-powered knowledge-base and knowledge-graph agent platform with RAG retrieval, document management, permission configuration, model configuration, and agent chat workflows.

## Quick Start

```bash
git clone git@github.com:96980754/shanli.git
cd shanli
docker compose up -d
```

## Highlights

- Knowledge base and document management
- Enterprise users, departments, and knowledge-base permissions
- RAG retrieval and question answering
- Knowledge graph and evaluation capabilities
- Web-based administration workspace
